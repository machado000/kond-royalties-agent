# Handoff

Esta pasta registra o contexto necessario para continuar o trabalho em
outra sessao.

Arquivos principais:

- `project-status.md`: estado atual do projeto e o que ja funciona
- `architecture-notes.md`: decisoes de arquitetura e limites da V1
- `validated-commands.md`: comandos que ja foram executados e validados
- `next-session.md`: instrucoes diretas para a proxima sessao
